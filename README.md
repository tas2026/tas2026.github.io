**TAS 2026: Trustworthy Agentic Systems**

A symposium in the AAAI 2026 Fall Symposium Series.
Web URL: https://tas2026.github.io/
Dates: November 5–7, 2026, Westin Arlington, Arlington, Virginia, USA

This site is built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme.

Local preview:
```
bundle install
bundle exec jekyll serve
```
Then open http://localhost:4000

To deploy on GitHub Pages, create a repository named `<username>.github.io` (or set the `url`/`baseurl` in `_config.yml`) and push these files. Replace `YOUR_USERNAME` in `_config.yml` with your GitHub username.

Items marked **TBA** / **TODO** in the pages should be updated as the program is finalized (keynote speakers, submission site, important dates, program committee, proceedings).
